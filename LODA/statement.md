# Project Statement: SmartSpend

## Problem Statement
Managing personal finances can be challenging without a proper tool. Many users struggle to keep track of their daily expenses, leading to overspending. Existing solutions are often too complex or require internet connectivity. SmartSpend aims to provide a simple, offline, and efficient way to track expenses.

## Scope
The application is a standalone desktop console application. It focuses on core personal finance features:
- User authentication to ensure data privacy.
- CRUD operations for expenses.
- Basic analytics to provide spending insights.
- Data persistence using local files.

## Target Users
- University students managing monthly allowances.
- Individuals looking for a simple, no-frills expense tracker.
- Users who prefer offline tools for privacy.

## High-Level Features
1.  **User Authentication**: Simple username/password login.
2.  **Expense Management**: Record date, category, amount, and description.
3.  **Budget Monitoring**: Alert users when spending exceeds predefined limits.
4.  **Category Reports**: Summarize spending habits.
